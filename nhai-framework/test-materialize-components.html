使用说明

// 1. 引入样式文件 <link rel="stylesheet" href="src/components/materialComponents/materialize.css"> <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet"> // 2. 创建组件 const button = NHAIObjectFactory.createMaterialButton('点击我') button.setVariant('raised') button.setColor('blue') button.setIcon('save') button.setOnClick(() => console.log('按钮被点击')) // 3. 渲染组件 const element = button.render() document.body.appendChild(element) // 4. 支持所有框架 // Vue: 在 mounted() 中渲染 // React: 在 useEffect() 中渲染 // Svelte: 在 onMount() 中渲染 // 原生 JS: 直接使用
