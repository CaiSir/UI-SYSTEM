💻 代码示例

基础使用
import { ModernNHAIButton } from 'nhai-framework'

// 创建按钮实例
const button = new ModernNHAIButton({
  type: 'primary',
  size: 'middle',
  children: '点击我',
  onClick: () => console.log('clicked')
})

// 渲染按钮
const element = button.render()
链接按钮
const linkButton = new ModernNHAIButton({
  type: 'link',
  href: 'https://example.com',
  target: '_blank',
  children: '外部链接'
})
路由按钮
const routerButton = new ModernNHAIButton({
  type: 'primary',
  href: '/home',
  router: (path) => {
    console.log(`Navigate to: ${path}`)
  },
  children: '首页'
})

📚 API 文档

属性 类型 默认值 说明
type 'primary' | 'default' | 'dashed' | 'link' | 'text' 'default' 按钮类型
size 'small' | 'middle' | 'large' 'middle' 按钮尺寸
children string - 按钮内容
disabled boolean false 是否禁用
loading boolean false 是否加载中
href string - 链接地址
target '_blank' | '_self' | '_parent' | '_top' '_self' 链接打开方式
onClick (event: Event) => void - 点击事件处理
router (path: string) => void - 路由处理函数

✨ 核心特性

🎨 声明式 API

采用声明式设计，通过 props 配置组件行为，使用简单直观

⚡ 性能优化

内置样式缓存机制，避免重复计算，提升渲染性能

🔗 链接支持

支持外部链接和内部路由，自动渲染为 a 标签或 button 标签

📱 响应式设计

支持多种尺寸和状态，适配不同设备和场景

🎯 TypeScript 支持

完整的类型定义，提供良好的开发体验和类型安全

🔄 状态管理

支持禁用、加载等状态，提供丰富的交互反馈
