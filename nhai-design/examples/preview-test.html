📋 测试说明

请按照以下步骤测试实时预览功能：

  1. 确保开发服务器正在运行 (http://localhost:3001)
  2. 打开浏览器开发者工具的控制台
  3. 访问 http://localhost:3001 并选择"在线编辑"模式
  4. 运行默认的测试代码
  5. 查看控制台输出和预览区域

🧪 测试代码

// 最简单的测试 console.log('开始测试...'); // 检查 NHAIObjectFactory console.log('NHAIObjectFactory:', typeof NHAIObjectFactory); if (typeof NHAIObjectFactory !== 'undefined') { // 创建按钮 const button = NHAIObjectFactory.createButton('测试按钮'); console.log('按钮创建成功:', button); // 设置样式 button.setVariant('primary'); // 渲染 const element = button.render(); console.log('渲染结果:', element); // 添加到预览区域 const previewArea = document.querySelector('.preview-area'); console.log('预览区域:', previewArea); if (previewArea) { previewArea.appendChild(element); console.log('组件已添加'); } else { console.error('找不到预览区域'); } } else { console.error('NHAIObjectFactory 未定义'); }
