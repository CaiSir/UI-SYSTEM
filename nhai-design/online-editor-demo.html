📝 使用示例

// NHAI 组件示例代码 const container = NHAIObjectFactory.createContainer() // 创建按钮 const button = NHAIObjectFactory.createButton('点击我') button.setVariant('primary') button.setWidth(120) button.setHeight(40) button.setOnClick(() => alert('按钮被点击！')) // 创建标签 const label = NHAIObjectFactory.createLabel('欢迎使用NHAI') label.setFontSize(16) label.setColor('#2c3e50') // 添加到容器 container.addChild(label) container.addChild(button) // 渲染到预览区域 const element = container.render() document.querySelector('.preview-area')?.appendChild(element)

🎯 如何使用

  1. 启动开发服务器：npm run dev
  2. 在左侧选择"在线编辑"模式
  3. 在代码编辑器中编写 NHAI 组件代码
  4. 点击"运行代码"查看预览效果
  5. 使用"语法检查"、"格式化"等功能优化代码

🛠️ 技术特性

  • Monaco Editor：VS Code 同款编辑器，提供专业级编辑体验
  • Vue 3 + TypeScript：现代化的前端技术栈
  • 安全沙箱：隔离的代码执行环境，确保安全性
  • 实时反馈：即时的语法检查和错误提示
  • 响应式设计：支持桌面和移动设备
