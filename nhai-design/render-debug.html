🔍 NHAI 渲染调试工具

诊断代码执行和渲染问题

📋 调试步骤

  1. 确保开发服务器正在运行 (http://localhost:3001)
  2. 打开浏览器开发者工具的控制台
  3. 点击下面的测试按钮
  4. 查看控制台输出和日志区域
  5. 检查预览区域是否显示组件

🧪 测试代码

// 调试版本的 NHAI 组件示例 console.log('开始创建 NHAI 组件...'); const container = NHAIObjectFactory.createContainer(); console.log('容器创建成功:', container); const button = NHAIObjectFactory.createButton('点击我'); console.log('按钮创建成功:', button); button.setVariant('primary'); button.setOnClick(() => { console.log('按钮被点击！'); alert('按钮被点击！'); }); container.addChild(button); console.log('按钮已添加到容器'); const element = container.render(); console.log('渲染结果:', element); const previewArea = document.querySelector('.preview-area'); console.log('预览区域:', previewArea); if (previewArea) { previewArea.appendChild(element); console.log('组件已添加到预览区域'); } else { console.error('找不到预览区域！'); }
